Office Deployment Scripts for IT Pros

Making Office 2016 and Office 365 ProPlus as simple as CLI

GitHub Script Repository

In an effort to streamline the deployment process for Office 2016 and Office 365 ProPlus, a GitHub respository has been created with a collection of scripts and other projects. The respository is meant to be a place for IT Pros to come and find solutions that will make tasks related to deploying and maintaining Office easier.

All items in the master branch of the respository are tested and ready to use. Each script has a ReadMe file that explains how to use it. The main ReadMe has a list of the name and descriptions for all of the scripts in the respository.

If you would like to contribute to respository please following the instructions
'How to Contribute to this project' in the main ReadMe file in the respository.
